# Meus Estudos de Linguagem R 📊

Este repositório contém meus exercícios e aprendizados em R, focados em manipulação de dados e visualização.

## 📂 Conteúdo
- **tabelas-test.R**: Exercícios de limpeza de dados com `dplyr` e o operador pipe (`%>%`).
- **Graficos-Basicos.R**: Exemplos de gráficos de dispersão e linhas.
- **Vendas-Sorvete**: Análise de correlação entre temperatura e vendas.

## 🛠️ Tecnologias
- R 4.5.2
- VS Code
- Tidyverse (ggplot2, dplyr)
